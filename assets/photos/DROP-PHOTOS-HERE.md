# Drop real photos here

The site is wired to **automatically use real photos** the moment you drop them
into this folder with the exact filenames below. No code changes needed — the
placeholder illustration is replaced on load.

TexAna's best food + truck photos live on **Facebook / Instagram**, which are
token-blocked and cannot be downloaded programmatically. So drop them here by hand.

## Filenames the site looks for

| Save as (in this folder)   | Shows up as                                   |
| -------------------------- | --------------------------------------------- |
| `hero-taco.jpg`            | Hero card — a signature birria / taco close-up |
| `truck.jpg`                | Story section — the truck parked at an event   |

## How to get them

1. Open the business's Facebook or Instagram
   (facebook.com/texanataceauxs · instagram.com/texana_taceauxs).
2. Save the best food + truck shots to your computer.
3. Rename them to match the table above and drop them in this folder.
4. Reload the site — real photos appear automatically.

Recommended: landscape ~1600px wide, JPG, well-lit. The hero card is square-ish;
the story frame is roughly 16:11.
